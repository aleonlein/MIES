File MIES_ExperimentConfig.ipf¶

ExpConfig Import user settings to configure paramters for Ephys experiments

These include:

  • Amplifier settings
  • Pressure regulator settings
  • Interactions with MCCs
  • DAEphys panel settings

Functions

variable ExpConfig_ConfigureMIES(variable middleOfExperiment = defaultValue)¶

Configure MIES for experiments.

Parameters
  • middleOfExperiment: [optional, defaults to false] Allows MIES config in the middle of experiment. Instead of setting MCC parameters they are pulled from actively recording MCCs to configure MIES]

static variable ExpConfig_Amplifiers(string panelTitle, wave UserSettings, variable midExp)¶

Open and configure amplifiers for Multi-Patch experiments.

Parameters
  • panelTitle: Name of ITC device panel
  • UserSettings: User settings wave from configuration Notebook
  • midExp: Configure in middle of experiment, default = 0
